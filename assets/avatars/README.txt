Drop avatar PNG files here. Recommended size: 256x256 or larger, square, transparent background.

Required filenames (matches avatar IDs in server/avatars.js):
- geckot.png    (Geckot     — 200 crystals, rare)
- lucario.png   (Lucario    — 200 crystals, rare)
- skepoke.png   (Skepoke    — 200 crystals, rare)
- spacechu.png  (Spacechu   — 200 crystals, rare)
- goldmew.png   (Gold Mew   — 250 crystals, epic)
- goldt.png     (Gold T     — 500 crystals, legendary)

If a file is missing, the avatar card will show a broken-image icon. Just drop it
in and refresh the browser — no server restart needed for image changes.

Server restart IS required when changing the catalog (server/avatars.js).
